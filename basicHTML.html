HTML Basics

Hey there, and welcome to CodePoint's basic HTML tutorial. For those of you that don't know, HTML stands for HyperText Markup Language. It's alright if you're not quite sure what that means, because we will be going over that later. First things first. A lot of people have the impression that code is very hard to read. While this may be true for certain programming languages, it is not the case with HTML. HTML consists of tags, and just about everything between tags is text - the text that is displayed on your browser. So how does it work? Let's take a look. (Screenshots are taking using Visual Studio Code, but you may use any text editor you would like)

Tags & Setup Code

There are two types of tags: paired tags and singular tags. Paired tags have an opening tag and a closing tag, whereas singular tags don't have a closing tag. Opening tags and singular tags consist of an opening angle bracket <, some text (usually a letter or two, but sometimes more), and a closing angle bracket >, and a closing tag consists of an opening angle bracket <, a forward slash /, the text from the opening tag, and a closing angle bracket >. Let's take a look at some basic tags.

The first tag we're going to look at is a singular tag, the <!DOCTYPE> tag. We always put this tag at the very beginning of an HTML document, to let the browser know that this document is written in HTML. To begin your first HTML document, write <!DOCTYPE html>. Now, we need to tell the browser that our HTML is starting. We do this with the <html> tag. <html> is a paired tag, so we also need to write </html>. All of our code will go between these two tags. We're almost done with setup. The last thing we need to do is add a <head> paired tag along with </head>, and a <body> along with a </body>. Your code should now look like this:

<!DOCTYPE html>
<html>
    <head></head>
    <body></body>
</html>

You'll notice that if you save your document, it defaults to a .txt file. While this is fine, you will have no way of seeing what your website looks like. Instead, while you name your document, make sure it ends in .html. This way, your file will look like the icon of your preferred browser, and when you run it, you will see what your browser looks like. (To edit your file, open your code editor, click File > Open File and select your document. Note: we haven't done anything but setup, so your webpage is going to be blank)

Adding Basic Content

So you now have a blank webpage. Pretty boring, right? Let's add some basic text: A heading, and a paragraph. You can put whatever text you like between the tags; the example uses some lorem ipsum dummy text.

To add a heading to your webpage, you can use the h tags. There are four default h tags: <h1> <h2> <h3> and <h4>. You can take your pick, <h1> has the largest font-size, and <h4>has the smallest. Let's add a header to the webpage, between the <body> and </body> tags. (For the purposes of this basic tutorial, just ignore the <head></head>. We will talk more about that and its uses later. Note that using any h automatically creates a new line underneath it.)

<!DOCTYPE html>
<html>
    <head></head>
    <body>
        <h1>This is a heading (h1)</h1>
    </body>
</html>

Now let's add a paragraph. Unlike the h tags, there is only one p tag: <p>

<!DOCTYPE html>
<html>
    <head></head>
    <body>
        <h1>This is a heading (h1)</h1>
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
    </body>
</html>

Now, save your document and reload. You should see text on your website! (It will be in your browser's default font, and the color will be black. We will learn how to change that later.)

Basic Formatting with HTML Tags

Most everything we do to format our webpages is done in CSS, but there are a couple basic things you can do with HTML, like center the text on your page, and add a new line. Let's take a look at some basic ones:

New Line (Break)

You may have noticed that adding a new line in your code doesn't do anything. Instead we have to use a singular tag: <br>. To create a new line, you just put a <br> where you want it. (<br> is a singular tag, so you don't need a closing </br> or anything. Using <p> after a previous </p> or </h(1,2,3,4)> automatically creates a new line.)

Text Formatting

While you can use CSS to add italic and bold text to your HTML, you can also use HTML tags! To get italic text, you can use the <em> tag, and for bold text use <strong>. There's also <u> for underline and <del> for strikethrough. (All of these tags are paired tags. You put the text you want formatted between the opening and closing tags.) Here is an example, using the code from above:

<!DOCTYPE html>
<html>
    <head></head>
    <body>
        <h1>This is a heading (h1)</h1>
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
        <p>
        <em>This is italicized text</em>
        <br>
        <strong>This is bold text</strong>
        <br>
        <u>This is underlined text</u>
        <br>
        <del>This is text with strikethrough</del>
        </p>
    </body>
</html>

Congratulations. You can now personalize your website with some basic text formatting!
